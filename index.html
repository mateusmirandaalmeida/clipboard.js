Instalação NPM:


Importe o arquivo JavaScript:
<script src="node_modules/clipboard.js/dist/clipboard.min.js"></script>

Exemplo por HTML:

HTML 
<button class="btn btn-default" 
        clipboard="Esse é um texto que foi copiado como exemplo | HTML"
        clipboard-success="alert('copiado')"
        clipboard-error="alert('erro')">
    Copiar
</button>

Exemplo por JavaScript:

HTML 
<button class="btn btn-default" 
        onclick="copiar()">
    Copiar
</button>
JavaScript 
function copiar() {
    var success = function () {
        alert('copiado');
    }
    var error = function () {
        alert('erro');
    }
    Clipboard.copy('Esse é um texto que foi copiado como exemplo. | JavaScript', success, error);
}
